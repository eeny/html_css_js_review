submit 버튼은 form 태그 안에 있는 input의 value를 백엔드로 넘겨주기 위해 있는 버튼이다.

        반드시 form 안에 있어야 하며, form 안에 있는 모든 input은 name을 가져야 한다.
        name이 자바에서 마치 변수와 같은 녀석이며, 변수값은 value에 담는다.
        데이터의 타입은 100% 문자열이다!

         <= 이렇게 작성했을 경우
        String id = null; <= 이것과 동일한 말이 된다.
        문자 입력을 할 수 있는 input 이므로 입력된 글씨가 value가 된다.
