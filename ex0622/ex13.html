    - body가 아닌 head에 쓴다. 설정에 대한 태그!
    - 닫는 태그가 없다!

    주로, 사이트에 대한 설명들을 작성한다.
    keywords / description / author / generator / application-name
    - keywords : 검색에 사용된다.(지금은 사용X - 무분별한 광고 때문에)
    - description : 사이트에 대한 설명
    - author : 만든 사람
    - generator : 만든 툴 - 우리는 Visual Studio Code
    - application-name : 이 웹프로그램의 이름
